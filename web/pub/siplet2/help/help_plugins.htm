Plugins

An Plugin is a collection of Timers, Aliases,, Triggers, and a special javascript Script that can receive and respond to special events by performing Actions.

Like other editors, you can add or remove global plugins from the Global Options/Plugins menu, or plugins that apply only to the active mud window's phonebook entry through the Window/Plugins menu.

The top list field in the Plugin editor allows you to either load a new plugin from a url, paste one into the text area, or delete an existing one from the list. Consult the Plugin help for the proper syntax of the add plugin text area.
